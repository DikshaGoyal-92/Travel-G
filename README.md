# Travel-G Website

Travel-G is a responsive travel and tourism website developed using HTML, CSS, and JavaScript.

## Features

- Responsive navigation bar
- Explore popular travel destinations
- Best travel offers section
- Trip booking form
- Contact form
- Login popup
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Sections

- Home
- Best Locations
- Best Offers
- Book Your Trip
- Contact Us
